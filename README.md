Design and develop a three-tier web application that allows you to register for an appointment with a doctor.
Operations available to the user:
1. Sign in
2. Register for an appointment (open doctor schedule, choose suitable day and time)
3. Get an appointmetnt details by email

Operations available to the doctor:
1. Sign in
2. View all appointments
3. View an appointment details
4. Cancel an appointment

Front-end is based on ReactJs. Back-end - .Net Core, EF.

Assumptions:
1. Simple styles for the client part
